# Raspberry-Pi-Security-Alarm-System
A simple security alarm system to detect intruders using a Raspberry Pi 4. 

## Tools and technologies

### Hardware 
- Raspberry Pi 4
- Micro SD Card
- A Passive Infrared Sensor (PIR) 

### Software 
- Python version: 3.11.
